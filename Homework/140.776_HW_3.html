Taking a look at the released column, it seems that some of the release dates have some extra unnecessary information in the format (US/UK)[##].

glimpse(raw_sales$released)
##  chr [1:48] "October 24, 2006" "November 11, 2008" "November 11, 2008" ...

This can be generalized as anything between the start parentheses and end bracket: (.]. To remove this, we use a regular expression, then transform it into the date format:

sales <- raw_sales %>% 
  mutate(released = str_remove_all(released, " \\(.*\\]") %>% 
                    mdy(.))

# we can also use the more specific regex " \\(\\w{2}\\)\\[\\d{2}\\]"

Next, lets look at the newest and oldest albums, as well as the median time between album releases.

sales %>% 
  filter(country == "US") %>% 
  group_by(artist) %>% 
  mutate(
    years_since_release = interval(lag(released), released) %>% 
                          as.duration(.) %>% 
                          as.numeric(., "years") %>% 
                          floor()
  ) %>% 
  summarize(
    oldest = year(min(released)),
    most_recent = year(max(released)),
    med_years_since_release = median(years_since_release, na.rm = TRUE)
  )
## # A tibble: 2  4

2E

Using the afinn lexicon, lets calculate the average sentiment for Taylor Swifts albums.

ts_avgsent <- ts_lyrics %>% 
  unnest_tokens(word, Lyrics, token = "words") %>% 
  anti_join(stop_words) %>% 
  group_by(Album) %>% 
  count(word) %>% 
  inner_join(get_sentiments("afinn")) %>% 
  summarize(
    avg_sentiment = sum(n * value) / sum(n)
  )

From the graph above, we can see a generally negative trend of average sentiment score over time. This means that Taylor Swifts lyrics have (on average) become more negative over time. The most notable exception to this trend is her most recent album with sales information, Lover, which has a much higher average sentiment score than her previous album.
